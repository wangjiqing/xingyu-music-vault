# v1.1.1 Release Notes — 安全边界文档与部署风险提示

**发布日期：** 2026.06.08

v1.1.1 是文档与部署风险提示版本，只补充安全边界说明，不实现任何认证、Token、登录、反向代理或接口逻辑变更。

## 版本边界

当前版本优先面向本地 / 局域网可信环境使用。不建议在未配置认证、HTTPS、反向代理保护的情况下，直接将服务暴露到公网。

当前说明为安全边界提示，不代表登录、完整 OpenAPI Token、HTTPS、反向代理模板或公网安全部署方案已经实现。

## 新增 / 补强

- README 增加安全提醒入口
- 部署文档增加本地 / 局域网优先的安全边界说明
- OpenAPI 文档增加公网暴露风险说明
- Docker Compose 示例增加端口暴露与安全配置预告
- 明确后续 v1.1.x 将提供管理员账号初始化、登录认证、OpenAPI Token、反向代理建议和 HTTPS / 公网暴露风险说明

## 后续 v1.1.x 规划

- 管理员账号初始化
