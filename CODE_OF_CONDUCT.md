# Code of Conduct

Xingyu Music Vault aims to be a friendly, practical open source project.

Please:

- Be respectful and patient.
- Assume good intent, and ask for clarification when something is unclear.
- Keep discussions focused on the project and its users.
- Avoid harassment, personal attacks, or publishing private information.
- Help keep issues and pull requests useful for future readers.

Maintainers may edit, close, or remove comments and contributions that make collaboration unsafe or unproductive.
