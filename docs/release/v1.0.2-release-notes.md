# v1.0.2 Release Notes

**主题：** 秋日唱片主题素材接入

v1.0.2 是一次主题素材接入版本，重点完成「秋日唱片 / Autumn Vinyl」作为第二套四季主题候选资源的落库与文档展示。本版本继续复用 v1.0.1 的主题结构，提供浏览器本地轻量切换入口，不引入服务端主题管理系统。

## 更新内容

- 新增 `frontend/public/themes/autumn-vinyl/` 主题资源目录
- 接入 README banner、Logo、favicon、小图标、背景图、页面空状态插画和主题色板
- 新增 Autumn Vinyl 主题候选配置，Theme ID 为 `autumn-vinyl`
- 当前默认主题仍为 `midsummer-starlight`
- 管理后台 header 右侧新增主题切换入口，选择结果写入浏览器 `localStorage`
- 主题切换时同步更新页面皮肤、空状态图、favicon 和 `theme.css`
- 前端当前主题资源路径集中到轻量配置 helper，便于后续 Winter Moonlight / Spring Dawn 复用
- `/api/open/v1/server/info` 的 `serviceVersion` 更新为 `1.0.2`
- Vite 构建排除所有主题目录下的 `archive/` 备份目录
- README、changelog、roadmap 和主题说明文档同步更新

## 版本边界

本版本不包含：

- 后端业务逻辑修改
- OpenAPI 契约变更
- 数据库结构变更
- 音乐扫描、元数据、歌词、封面等业务能力扩展
- Winter Moonlight / Spring Dawn 接入
- 服务端主题管理系统

## 资源目录

```text
frontend/public/themes/
  midsummer-starlight/
  autumn-vinyl/
```

## 已知限制

- Autumn Vinyl 当前为生成素材工程化版本，还不是最终品牌资产
- 背景图不是原生 4K 终稿
- Logo 来源于展示板裁切，后续建议使用原始 SVG / 分层文件规范化输出
- 空状态图建议后续按页面语义重绘

## 验证建议

```bash
cd frontend
npm run build
```
