# v1.2.2 Release Notes — 歌曲工作台小屏布局修复

v1.2.2 聚焦管理端「歌曲工作台」的小屏与低高度显示体验，修复播放器、歌词页、封面页、元数据页和 OpenAPI 输出页在窄内容区下溢出或被裁切的问题。本版本不改变后端业务接口、OpenAPI 契约或数据库结构。

## 改进

- 歌曲工作台引入页面级缩放变量，小屏或低高度环境下整体降低布局密度。
- 播放器改为按组件自身宽度响应，播放按钮、进度条、时间、播放模式和音量控件会随工作台比例缩小。
- 修复音量滑块在右侧满格时手柄探出播放器边界的问题。
- 歌词页右侧封面、歌词字号、行距和歌曲信息跟随工作台比例缩放。
- 封面页图片区域和详情区域跟随工作台比例缩放。
- 元数据和 OpenAPI 输出 Tab 支持内容区滚动，避免长内容被裁切。
- OpenAPI JSON 预览区域跟随工作台比例缩放。
- image 模式 Docker Compose 默认镜像更新到 `v1.2.2`，封面目录挂载目标与 `APP_ARTWORK_SCAN_DIR` 保持一致，避免容器内封面路径不匹配导致页面提示封面缺失。

## 版本同步

- `/api/health` 返回版本更新为 `1.2.2`。
- `/api/admin/server/info` 的 `serviceVersion` 更新为 `1.2.2`。
- `/api/open/v1/server/info` 的 `serviceVersion` 更新为 `1.2.2`。

## 非目标

本版本不包含确认状态、确认按钮、AI 候选、网络刮削、歌词公网请求、音频 Tag 写回、封面写入音频文件、复杂播放队列、播放历史、播放统计或收藏系统。

- 后端版本号测试断言已同步到 `1.2.2`。
