# Xingyu Music Vault / 星语音库

> 音乐元数据管理后台系统，面向个人音乐库、NAS、家庭服务器、自托管环境。

## 项目定位

Xingyu Music Vault 是一个**后台元数据管理系统**，而非播放器。它负责：

- 扫描音乐文件，提取元数据
- 清洗、确认、补充元数据（歌手、专辑、封面、歌词）
- 提供 API 供客户端消费
- 支持自托管环境下的私有部署

## 核心功能

- [ ] 音乐库扫描与指纹识别（规划中）

## 技术栈

| 层级   | 技术选型                |
|--------|------------------------|
| 后端   | Quarkus (Java 21)      |
| 前端   | Vue 3 + TypeScript      |
| 数据库 | SQLite（开发）/ PostgreSQL（生产） |
| 存储   | 本地文件系统            |
| 部署   | Docker Compose         |
